RWD/FLEXBOX筆記

RWD
目的:將電腦橫的顯示，改成手機的顯示
步驟
  • 先在head中加上meta name="viewport" content="device-width, initial-scale=1, maximum-scale=1" /(固定縮放)
  • 然後在style加上css的madia query，@media screen and (max-width:600px){.content{width:100%;}}(.content表其父輩)
*其他字形大小或置中等，皆加在media query中即可
*使用開發人員開發工具可察看手機版的狀況

FLEXBOX
目的:讓容器中的項目可以並排(切欄)顯示，以及水平對齊和垂直對齊
步驟
  • 先將容器的style加上display:flex;項目加上flex:none;width:300px;margin:10px;(代表項目無法自動縮放)
  • 在容器的style加上justify-content:center;(使內容置中)
CSS3 Flexbox 回應式設計 RWD 排版
目的:欄位變化
範例: 4欄>2欄>1欄
規劃:
1200px以上:4欄，固定尺寸
500px-1200px:2欄，彈性尺寸
500px以下:1欄，彈性尺寸
步驟:
  • 將style加上@media (max-width:1200px){ main>.item{ width:45%; } }(留下一些margin的空間)
  • 在容器的style加上flex-wrap:wrap;(使容器中的項目換行)
  • 然後在style加上@media (max-width:500px){ main>.item{ width:90%; } }
*須按照順序，否則須添加更多設定
